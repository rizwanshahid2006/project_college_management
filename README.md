# 🎓 ITM College Management System

## 📌 Introduction

In today’s digital world, most manual paperwork such as accounting, record keeping, admissions, and result declaration has become computerized. Educational institutions are increasingly using websites to manage their operations efficiently.

The **ITM College Management System** is a web-based application developed to provide an online platform for students and administrators. The main purpose of this system is to present college information, manage admissions, maintain student records, and allow administrators to update the website content.

This website helps the college to promote its achievements, provide course information, and allow students to apply for admission online.

---

# 🎯 Project Objectives

* Provide complete information about the college.
* Allow students to register and apply for admission online.
* Maintain records of students digitally.
* Allow admin to manage website data.
* Display courses, departments, faculty, and facilities.

# 📝 Online Admission

Students can apply for admission through an online admission form.

Students can:

* Download admission form
* Fill admission form online
* Submit required details

After submitting the form, the system generates a **Unique Registration Number** which can be used to check the admission status.

Admission Status:

* Waiting
* Accepted
* Rejected

If the admission is **accepted**, the student can proceed to complete additional details like subject selection.
